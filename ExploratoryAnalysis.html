Change ORIGIN_AIRPORT_ID to a string¶

ORIGIN_AIRPORT_ID is an integer representing an airport. Using information provided by the BTS, we convert the ID into a string representation. For example 101010 => Los Angeles International

In [3]:
"""Converts all the airport id's into the airport's name"""

idToAirport = {}  # Dictionary holding id to airport

with open('L_AIRPORT_ID.csv_', 'r') as f:
    reader = csv.reader(f)

    for i, row in enumerate(reader):

        # .csv file holds name of airport and city, so split
        if i != 0:
            id = int(row[0])
            airport = row[1].split(': ')[-1]

            # Add to dict
            if id not in idToAirport:
                idToAirport[id] = airport

data['ORIGIN_AIRPORT_ID'] = data['ORIGIN_AIRPORT_ID'].map(idToAirport)

"""Get the percentage of delayed flights at each airport"""

delayRate = []
flightCounts = data['ORIGIN_AIRPORT_ID'].value_counts()

# Iterate through the df and calculate delay rate with flightCounts
for i, row in sortedDelayCount_df[:n].iterrows():
    airport = row[0]
    delay = row[1] * 1.0 / flightCounts[airport]
    delayRate.append((airport, delay))
    
delayRate.sort(key=lambda tup: tup[1], reverse=True)

# Plot result
delayRate_df = pd.DataFrame(delayRate)
ax = delayRate_df[:n].plot.barh(figsize=(16, 9), fontsize=15)
ax.set_yticklabels(delayRate_df[0])
plt.title('Popular Airport Delay Rate')
ax.title.set_size(15)
ax.invert_yaxis()

This plot shows us the top 15 busiest airports ranked by their rate of delay. At the top, we see the infamous Los Angeles International, with over 50% of their flights being delayed. One interesting thing to note from this plot is Hartsfield-Jackson Atlanta International, which is the busiest airport in the United States, yet, only just over 40% of their flights are delayed.

How many airports account for 50%, 75%, 90% of flights in the U.S?¶

In [14]:
numFlights = len(data)
airports = []
total = 0
percentages = [.5, .75, .90]
all_airports = {}

for p in percentages:
    total = 0
    airports = []
    
    # flightCounts is sorted by count, so sum count until 50%
    for a, c in flightCounts.iteritems():
        airports.append(a)
        total += c

        if total >= p * numFlights:
            break
    
    all_airports[p] = airports
    
    print('{:.0f} airports account for {:.2f}% of the flights in the U.S'.format(len(airports), 100 * total / numFlights))

print('There are {} unique airports in this dataset'.format(len(flightCounts)))

From this plot, we observe that Saturdays have the fewest number of flights. In fact it's a staggering 18% drop from Friday. One possible theory to explain this is that business people tend to finish their business travels by the end of the week so that they can be home for the weekend. According to Airlines.org, about 31% of travellers travel for business, suggesting some evidence for this theory. (http://airlines.org/wp-content/uploads/2016/04/2016Survey.pdf)
